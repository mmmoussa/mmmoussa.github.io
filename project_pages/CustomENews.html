Custom eNews

  • A script for keeping up to date with the latest news from CBC.ca that interests you
  • The script parses CBC's RSS feeds, searching to see if any words from a list you've made are making headlines, and if any are, you are sent an email using the SendGrid API
  • This project can be set up with a cronjob on a server so that you are always up-to-date on what matters to you most
  • This project contains documentation for setting it up on a QNAP NAS server
