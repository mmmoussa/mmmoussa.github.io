My projects

Ramadan Hub

  • For this project, I created a website where people could go to find resources they'd need to facilitate their fasting during the Muslim month of Ramadan
  • The website served as a hub where people could find daily prayer times, local events and restaurants, and mosque locations
  • The website also contains an easily readable full copy of the Quran in both English and Arabic
  • For this project I used tools including HTML, CSS, Javascript, Python, Bootstrap, JSON, and the Django framework
  • In order to create this project, I had to create and use custom scripts to scrape websites as well as to collect and organize api data
  • Once the website was created, I visited local restaurants and collected money in exchange for advertisement on the website, and the money was then donated to provide food at free dinners held by the UWMSA
  • This project can be viewed at ramadanhub.com
  • The website had approximately 900 unique visitors during Ramadan (one month)
